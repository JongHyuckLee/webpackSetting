# webpack setting 
- webpack test server & build 기본 환경 구성 프로젝트

# package install
- npm install

# start test server
- npm run start:dev

# build
- npm run build

# support
- HRM
- css-loader
- babel-loader

# MiniCssExtractPlugin
- js 파일 별로 필요한 css파일을 번틀링

# HtmlWebpackPlugin
- 자동으로 js 및 리소스를 entry point html에 삽입

# CleanWebpackPlugin
- 빌드 시 빌드 디렉토리 정리 후 재 빌드

# entry point
- js : src/main.js
- html : public/index.html

# local server
- http://localhost:5500/
